# Contributing

This handbook reflects the collective experience of the CaRCC AI Facilitation
Materials Working Group. It is not a finished product — corrections,
clarifications, and new material from the broader RCD community are welcome and
expected to keep it current as tools and practices evolve.

Before contributing, please read the [Introduction](index.md) to understand the
handbook's scope and the AI project lifecycle framework it is organized around.

## Technical Infrastructure

The site is built with [Material for
MkDocs](https://squidfunk.github.io/mkdocs-material/) from plain Markdown files
in the [`handbook/`
directory](https://github.com/CaRCC/CaRCC-AI-Facilitation-Handbook/tree/main/handbook)
of the repository, and published to GitHub Pages. If you have not written
Markdown before, GitHub's [basic writing and formatting
guide](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)
covers the syntax you will need.

## How to Contribute

**If you are comfortable with GitHub:**

1. Fork the
   [CaRCC-AI-Facilitation-Handbook](https://github.com/CaRCC/CaRCC-AI-Facilitation-Handbook)
   repository.
2. Edit the relevant file(s) under `handbook/` — either locally, or directly in
   the GitHub web editor.
3. Preview your changes locally with `mkdocs serve` (see the
   [README](https://github.com/CaRCC/CaRCC-AI-Facilitation-Handbook#development)
   for setup), or use GitHub's built-in Markdown preview.
4. Open a pull request describing what you changed and why.

**If you are less familiar with git/GitHub:**

- Open a [new
  issue](https://github.com/CaRCC/CaRCC-AI-Facilitation-Handbook/issues/new)
  describing your suggestion, correction, or addition. This requires a free
  GitHub account but no git experience.

When adding a citation, add the corresponding entry to
[`refs.bib`](https://github.com/CaRCC/CaRCC-AI-Facilitation-Handbook/blob/main/refs.bib)
and reference it inline using the same `mkdocs-bibtex` citation syntax used
throughout the rest of the handbook.
